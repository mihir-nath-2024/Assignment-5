What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, where each node in the tree corresponds to a part of the document, such as an element, attribute, or text.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript that allows you to manage event listeners efficiently, particularly when dealing with a large number of elements that may not yet exist when the page initially loads. Instead of adding event listeners to individual elements, you delegate the responsibility to a parent element, leveraging the concept of event bubbling.
